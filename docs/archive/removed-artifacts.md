# Removed Runtime and Draft Artifacts

Phase 0 removed tracked files that were generated by local runs, private user
configuration, temporary product design drafts, or agent tooling.

Removed categories:

- `user_profile.json`, `user_config.json`, `keywords_config.json`, and
  `my_scholars.json`: local user configuration and preferences.
- `cache/**/*.json`: runtime recommendation caches, feedback, favorites, journal
  caches, and recommendation run outputs.
- `history/digest_*.md` and `daily_arxiv_digest.md`: generated digest outputs.
- `Paper Recommend.zip` and `Paper Recommend/`: divergent design/package drafts.
- `progress.txt`, `prd.json`, and `scripts/ralph/`: temporary agent planning
  artifacts.

These files were removed from source control rather than archived wholesale so
future agents and reviewers only see the canonical runtime source tree. If a
specific removed artifact is needed for historical investigation, restore it from
git history before this cleanup commit.
